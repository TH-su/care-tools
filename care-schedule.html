    ☁️ クラウド同期（Google スプレッドシート）

    Google スプレッドシートにデータを保存し、複数端末で共有できます。 初回はセットアップが必要です（下の「セットアップ手順」を参照）。

    入力すると自動保存されます（端末を閉じても、次回起動時にURLは保持されます）。
    ※ ワークスケジュールで同期設定済みの場合は、その接続先が自動的に使われます（ここは空でOK）。
    📋 セットアップ手順（初心者向け）

    所要時間：約10分

    1. Google スプレッドシートで 新規シート を作成
    2. メニュー「拡張機能 → Apps Script」を開く
    3. 表示されたエディタの中身を全部消し、下の「📜 Apps Script コード」をコピペ
    4. 💾 保存（Ctrl+S）→「デプロイ」→「新しいデプロイ」
    5. 歯車⚙ →「ウェブアプリ」を選択
    6. 「次のユーザーとして実行: 自分」「アクセスできるユーザー: 全員」を選択 →「デプロイ」
    7. 初回はアクセス権限を求められます。「許可」して進む（警告画面が出たら「詳細」→「安全でないページに移動」）
    8. 表示された ウェブアプリ URL をコピー → 上の「Apps Script Web App URL」欄に貼り付け
    9. 「設定を保存」→「接続テスト」で OK と出れば完了
    10. 「⬆️ クラウドへ保存」で初回アップロード

function doPost(e) {
  return _handle(e, 'POST');
}
function _handle(e, method) {
  try {
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName('Data');
    if (!sheet) {
      sheet = ss.insertSheet('Data');
      sheet.getRange('A1').setValue('');
      sheet.getRange('B1').setValue(0);
    }
    if (method === 'POST') {
      var body = JSON.parse(e.postData.contents);
      if (body && body.data !== undefined) {
        sheet.getRange('A1').setValue(body.data);
        sheet.getRange('B1').setValue(new Date().getTime());
      }
    }
    var data = sheet.getRange('A1').getValue();
    var ts   = sheet.getRange('B1').getValue();
    return ContentService
      .createTextOutput(JSON.stringify({ok:true, data:String(data||''), timestamp:Number(ts||0)}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (err) {
    return ContentService
      .createTextOutput(JSON.stringify({ok:false, error:String(err)}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
